Live For Speed

Live for Speed:It's a game that's focused on providing a realistic racing experience,with an emohasis on multiplayer gameplay.Players can race against each other Online,or against AI-controlled cars in single player mode.The game is know for its attention to details,with realistic physics and graphics that make it feel like you are really behind the wheel of a race car.The official website for the game is liveforspeed.com,and it provides information about the game ,including its history,features,and community.
